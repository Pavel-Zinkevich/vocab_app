Vocab App — Minimal Flutter example

This is a minimal Flutter app demonstrating a 3-tab layout using a BottomNavigationBar.

Tabs:
- Vocabulary — shows a list of words, has an AppBar and a + FAB to add words in-memory.
- Training — placeholder "Training coming soon".
- Profile — placeholder "Profile coming soon".

How to run

1. Ensure Flutter SDK is installed and on your PATH: https://flutter.dev/docs/get-started/install
2. From the project root (where `pubspec.yaml` is located) run:

```bash
flutter pub get
flutter run
```

Notes

- This app uses an in-memory list (no persistence).
- Code is intentionally simple and beginner-friendly.
